# KANO 图表模板说明

## 图表设计规范（手绘学术风格）

来自 `d:\cursor新尝试\kano\kano_chart.html` 的实战设计，每次生成新图表时参照此规范。

### 视觉风格

| 属性 | 值 |
|------|---|
| 背景色 | `#F9F6F0`（暖米色纸张感） |
| 图表区背景 | `#FFFDF8`（象牙白） |
| 字体-中文 | `'SimSun', '宋体', serif` |
| 字体-英文/数字 | `Arial, sans-serif` |
| 边框风格 | `1.5px dashed #d4cfc6`（虚线，手绘感） |
| 手绘滤镜 | SVG `feTurbulence` + `feDisplacementMap`（让边缘微微不规则） |

### SVG 象限图规范

- **viewBox**: `"0 0 960 540"`（扁长比例，便于截图不被截断）
- **坐标系中心**: x=480, y=275
- **轴线**: 虚线（`stroke-dasharray="6,4"`） + 贝塞尔曲线（手绘感）
- **象限底色**:
  - 左上（魅力型）: `#e8f5ec` 绿色调
  - 右上（期望型）: `#dfedf8` 蓝色调
  - 左下（无差异型）: `#f0f0ee` 灰色调
  - 右下（必备型）: `#fce8e6` 红色调

### 散点标注规范

每个功能点：
```
圆点（r=8，带白色描边 stroke-width=2.5）
  └── 引线（细线 stroke-width=0.7，opacity=0.4）
       └── 标签第一行：功能名（font-size=13, font-weight=700）
       └── 标签第二行：提及X次（font-size=9.5, fill=#999, class=en-txt）
```

各分类颜色：
| 类型 | 圆点颜色 | 象限 |
|------|---------|------|
| 必备型 | `#b5382a`（砖红） | 右下 |
| 期望型 | `#2563a0`（学院蓝） | 右上 |
| 魅力型 | `#1c7a42`（常青绿） | 左上 |
| 无差异型 | `#7a7a7a`（石灰灰） | 左下 |

### 标签防重叠策略

- 引线从圆点引出，标签放在引线终点
- 同一象限内，不同功能点的标签要**错开放置**（y 轴间距至少 30px）
- 靠边的点标签向内引（避免超出 viewBox 边界）
- 实测：右下象限（必备型）点多，将标签分左右两侧放置

### 象限水印文字

```svg
<!-- 中文名 opacity=0.35，英文名 opacity=0.28 -->
<text x="140" y="82" font-size="16" fill="#1c7a42" opacity=".35" font-weight="900">魅力型</text>
<text x="140" y="100" font-size="12" fill="#1c7a42" opacity=".28" class="en-txt" font-weight="700">ATTRACTIVE</text>
```

### 底部功能卡片

每张卡片包含：
- 顶部彩色细条（2.5px，对应各类型颜色）
- 功能名 + 分类徽章（右上角圆角标签）
- 功能描述（灰色小字）
- 访谈提及 X 次（Arial 字体）
- 核心证据（正文，含提及次数开头）
- 用户原话引用（左侧彩色边框，米色背景，斜体）

## 数据结构（生成图表所需）

```javascript
const F = [
  {
    n: "功能名",           // 显示名称
    t: "cm",              // 卡片类: cm/cp/ca/ci
    b: "bm",              // 徽章类: bm/bp/ba/bi
    tl: "必备型",          // 类型显示文字
    d: "功能描述",
    m: 1887,              // 提及次数（数字）
    e: "提及 X 次，核心分析...",  // 证据文字，以"提及X次"开头
    q: "「用户原话」"
  },
  ...
]
```

类型对应关系：
| KANO类型 | t值 | b值 |
|---------|-----|-----|
| 必备型 | cm | bm |
| 期望型 | cp | bp |
| 魅力型 | ca | ba |
| 无差异型 | ci | bi |
